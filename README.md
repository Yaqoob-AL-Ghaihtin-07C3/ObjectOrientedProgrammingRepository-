# البرمجة الكائنية بلغة C++ - أكواد وأمثلة عملية

يحتوي هذا المستودع على الأمثلة البرمجية العملية والشروحات والتمارين الخاصة بمقرر **البرمجة الكائنية (OOP) بلغة C++**. تم تنظيم الأكواد في مجلدات بناءً على رقم الفصل لضمان سهولة التصفح والاستخدام.

---

## هيكل المستودع

تم تقسيم المستودع إلى مجلدين رئيسيين لتنظيم المحتوى:

| المجلد | الوصف |
| :--- | :--- |
| `src` | يحتوي على الأمثلة البرمجية والشروحات المباشرة لكل فصل. |
| `exercises` | يحتوي على حلول التمارين العملية والواجبات لكل فصل. |

## كيفية الاستخدام

1.  **استنساخ المستودع:**
    ```bash
    git clone <رابط المستودع>
    ```
2.  **التصفح:**
    *   لتصفح الأمثلة البرمجية، انتقل إلى مجلد `src`.
    *   لتصفح حلول التمارين، انتقل إلى مجلد `exercises`.
3.  **التشغيل:**
    *   يمكنك تجميع وتشغيل ملفات C++ باستخدام أي مترجم (Compiler) يدعم C++، مثل GCC أو Clang.
    *   مثال على التجميع باستخدام GCC:
        ```bash
        g++ -o my_program my_program.cpp
        ./my_program
        ```
