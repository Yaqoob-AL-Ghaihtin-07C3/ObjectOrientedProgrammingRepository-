### شرح مفصل لكل تمرين:

#### 1. [Exercises5_1.cpp](Exercises5_1.cpp) - حساب مساحة دائرة
- **ما يفعله البرنامج**: يحسب مساحة دائرة باستخدام دالة خارجية
- **كيف يعمل**: 
  1. يعلن عن دالة `circarea` في الأعلى
  2. في `main()`، يطلب نصف القطر من المستخدم
  3. يستدعي `circarea(rad)` ويمرر نصف القطر
  4. الدالة تحسب: `المساحة = نصف القطر² × π`
  5. تعيد الدالة النتيجة لـ `main()` التي تعرضها
- **الغرض التعليمي**: تعلم أساسيات إنشاء واستدعاء الدوال، وفصل الوظائف المختلفة

#### 2. [Exercises5_2.cpp](Exercises5_2.cpp) - رفع عدد إلى أس
- **ما يفعله البرنامج**: يرفع عدداً إلى أس محدد، مع إمكانية استخدام قيمة افتراضية
- **كيف يعمل**: 
  1. يعلن عن دالة `power` بقيمة افتراضية للمعامل الثاني: `p=2`
  2. يطلب العدد من المستخدم
  3. يسأل إذا كان يريد إدخال أس محدد
  4. إذا نعم، يستدعي `power(number, pow)`
  5. إذا لا، يستدعي `power(number)` باستخدام القيمة الافتراضية
  6. الدالة تستخدم حلقة لضرب العدد في نفسه `p` مرة
- **الغرض التعليمي**: فهم القيم الافتراضية للمعاملات وكيف تجعل الدوال أكثر مرونة

#### 3. [Exercises5_3.cpp](Exercises5_3.cpp) - تعديل المتغيرات الأصلية
- **ما يفعله البرنامج**: يعدل المتغيرات الأصلية مباشرة باستخدام المراجع
- **كيف يعمل**: 
  1. يعلن عن دالة `zeroSmaller` تأخذ مراجع (`int&`)
  2. يهيئ أربعة متغيرات بقيم ثابتة
  3. يستدعي الدالة مرتين بمتغيرات مختلفة
  4. الدالة تقارن العددين وتجعل الأصغر يساوي 0
  5. لأنها مراجع، التغيير يطال المتغيرات الأصلية
- **الغرض التعليمي**: فهم الفرق بين تمرير بالقيمة وتمرير بالمرجع، ومتى نستخدم كل منهما

#### 4. [Exercises5_4.cpp](Exercises5_4.cpp) - هياكل كمعاملات للدوال
- **ما يفعله البرنامج**: يدير قياسات معقدة باستخدام هياكل ودوال
- **كيف يعمل**: 
  1. يعرف هيكل `Distance` للأقدام والبوصات
  2. يعلن عن دالتين: `bigengl` (ترجع القياس الأكبر) و `engldisp` (تعرض القياس)
  3. يطلب قياسين من المستخدم
  4. يستدعي `bigengl` لمقارنة القياسين وإرجاع الأكبر
  5. يستخدم `engldisp` لعرض القياسات بتنسيق `x'-y"`
- **الغرض التعليمي**: تعلم كيفية التعامل مع الهياكل في الدوال، سواء كمعاملات أو كقيم مرجعة
