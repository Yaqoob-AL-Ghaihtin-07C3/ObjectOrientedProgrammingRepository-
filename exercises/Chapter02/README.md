  2. يستخدم التخصيص المركب `var *= 2` لتصبح القيمة 20

- **الغرض التعليمي**: فهم مشغلات التخصيص المركبة (`*=`, `+=`, إلخ) والفرق بين الزيادة/النقصان السابق واللاحق
