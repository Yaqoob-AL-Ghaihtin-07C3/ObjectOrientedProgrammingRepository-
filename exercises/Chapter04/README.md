#### 1. [Exercises4_1.cpp](Exercises4_1.cpp) - تخزين وعرض أرقام الهواتف

- **كيف يعمل**:

  3. يطلب من المستخدم إدخال قيم لهيكل `ph2`
  4. يعرض كلا الرقمين بتنسيق: (رمز المنطقة) المركز-الرقم
- **الغرض التعليمي**: تعلم أساسيات تعريف الهياكل، تهيئتها، والتعامل مع أعضائها

#### 2. [Exercises4_2.cpp](Exercises4_2.cpp) - تمثيل وعملية نقاط على المستوى

- **كيف يعمل**: 
  1. يعرف هيكل `point` بمتغيرين: `xCo` (الإحداثي السيني)، `yCo` (الإحداثي الصادي)
  2. ينشئ ثلاث هياكل من النوع `point`: `p1`, `p2`, `p3`
  3. يطلب إحداثيات `p1` و `p2` من المستخدم
  4. يحسب `p3` = `p1` + `p2` عن طريق جمع الإحداثيات المناظرة
  5. يعرض إحداثيات `p3`
- **الغرض التعليمي**: فهم كيفية استخدام الهياكل لتمثيل كيانات رياضية وإجراء عمليات عليها

#### 3. [Exercises4_3.cpp](Exercises4_3.cpp) - حساب حجم غرفة باستخدام هياكل متداخلة
- **ما يفعله البرنامج**: يحسب حجم غرفة باستخدام نظام قياس مركب (أقدام وبوصات)
- **كيف يعمل**: 
  1. يعرف هيكل `Distance` لتمثيل المسافة بالأقدام (`feet`) والبوصات (`inches`)
  2. يعرف هيكل `Volume` الذي يحتوي على 3 هياكل `Distance` (للطول، العرض، الارتفاع)
  3. يهيئ هيكل `room1` بأبعاد ثابتة
  4. يحول كل بُعد إلى أقدام كسرية (بوصات ÷ 12)
  5. يحسب الحجم = الطول × العرض × الارتفاع
  6. يعرض النتيجة بالأقدام المكعبة
- **الغرض التعليمي**: تعلم مفهوم الهياكل المتداخلة وكيفية التعامل مع البيانات المعقدة المركبة
